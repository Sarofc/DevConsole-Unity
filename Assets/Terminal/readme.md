# DevConsole

> ## 特性

- 基于Unity的日志模块，可多线程接收。
- 模拟控制台程序执行命令。
  <br>实例/静态方法动态绑定，多参数支持。
  <br>命令缓存，命令补全。
- UGUI界面，支持调节窗口大小。
- Mobile、PC支持。

> ## 安装

- 使用UPM。 输入 https://github.com/Sarofc/DevConsole-Unity.git#package
